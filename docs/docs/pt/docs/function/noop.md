# noop

Função que não faz nada (no operation).

## Sintaxe
```typescript
noop(): void
```

## Parâmetros

| Nome | Tipo | Descrição |
|------|------|-----------|
| —    | —    | Nenhum.   |

## Retorno

| Tipo   | Descrição         |
|--------|-------------------|
| `void` | Não retorna nada. |

## Exemplo
```typescript
noop(); // undefined
```

## Notas
- Útil como função padrão, placeholder ou callback.
