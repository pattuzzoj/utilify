# noop

A function that does nothing (no operation).

## Syntax
```typescript
noop(): void
```

## Parameters
| Name | Type | Description |
| ---- | ---- | ----------- |
| –    | –    | None.       |

## Returns
| Type   | Description    |
| ------ | -------------- |
| `void` | Returns nothing. |

## Example
```typescript
noop(); // undefined
```

## Notes
- Useful as a default function, placeholder, or callback.
